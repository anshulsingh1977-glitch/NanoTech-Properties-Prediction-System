# NanoTech Properties Prediction System

A web-based platform for nanomaterial property prediction and analysis.

## Features

* Nanomaterial parameter input
* Property prediction dashboard
* Prediction history tracking
* CSV export functionality
* Sample management
* Responsive user interface

## Technologies Used

* HTML
* CSS
* JavaScript

## Future Improvements

* Machine Learning integration
* Real dataset support
* Flask/Python backend
* User authentication
